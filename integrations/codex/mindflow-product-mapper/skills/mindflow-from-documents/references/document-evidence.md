# Document evidence

Use compact locators: PDF `file.pdf#page=12`, Word/Markdown `file#heading`, workbook `file.xlsx#Sheet!A1:H40`, slides `deck.pptx#slide=8`, image `image.png#region=top-right`, URL `url#heading`. Save a paraphrased conclusion beside each locator. Do not retain complete extracted text, credentials, personal data, or hidden metadata.

Root and application narratives must be traceable to these compact conclusions. Evidence coverage is semantic rather than keyword based: record which source conclusions support each overview, goal, and application description, but do not require a fixed domain vocabulary or a fixed application count.
